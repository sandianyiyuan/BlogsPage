Spring IO

Spring IO 项目用于帮助管理项目依赖的版本号,创建Maven项目后在官网找到将其拷贝放入pom中即可。

其中管理了大量依赖，各个依赖版本之间正确可用。

Spring IO 已经停止维护，鼓励使用Spring Boot。
